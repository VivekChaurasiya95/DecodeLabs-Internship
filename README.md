## 🌟 Executive Summary
Welcome to the comprehensive repository for my **DecodeLabs Internship Tasks**. This workspace encapsulates my journey through a series of progressive data science and machine learning projects. Each project is designed to tackle real-world data scenarios, starting from foundational data exploration, advancing to complex feature engineering, and culminating in sophisticated supervised (classification) and unsupervised (clustering) machine learning implementations.

---

## 📂 Detailed Project Directory

### 1️⃣ [DecodeLabs Drive Task](./DecodeLabs%20Drive%20Task/)
**Basic Exploratory Data Analysis (EDA) on Customer Shopping Data**
* **Objective:** Establish a foundation in data handling by loading, inspecting, and summarizing a customer shopping dataset.
* **Techniques Used:** Basic Pandas functions (`.info()`, `.describe()`, `.isnull()`), data shape evaluation, and column type analysis.
* **Dataset:** 99,457 records containing invoice details, customer demographics (age, gender), product categories, pricing, and payment methods.

### 2️⃣ [DecodeLabs Task 1](./DecodeLabs%20Task%201/)
**Advanced EDA, Feature Engineering, & Data Preprocessing**
* **Objective:** Transform raw, messy E-commerce order data into a clean, model-ready format while extracting deep statistical insights.
* **Techniques Used:** Outlier detection (IQR method), Missing value imputation (KNNImputer), Feature generation (Datetime extraction), Categorical Encoding (OneHot), Scaling (StandardScaler), and Feature Selection (Random Forest Importance & Mutual Information).
* **Highlights:** Extensive use of Matplotlib and Seaborn for univariate, bivariate, and multivariate analysis.

### 3️⃣ [DecodeLabs Task 2](./DecodeLabs%20Task%202/)
**Credit Card Fraud Detection (Imbalanced Classification)**
* **Objective:** Build robust machine learning models to detect fraudulent credit card transactions, addressing the challenge of severe class imbalance.
* **Techniques Used:** Scikit-Learn Pipelines, `imblearn` for SMOTE (Synthetic Minority Over-sampling Technique), Hyperparameter tuning with GridSearchCV.
* **Models:** Logistic Regression, Random Forest Classifier.
* **Metrics:** Precision, Recall, F1-Score, ROC-AUC, and Confusion Matrices.

### 4️⃣ [DecodeLabs Task 3](./DecodeLabs%20Task%203/)
**Customer Segmentation & RFM Analysis (Unsupervised Learning)**
* **Objective:** Group an online retail store's customers into distinct segments to drive targeted marketing and business strategy.
* **Techniques Used:** RFM (Recency, Frequency, Monetary) calculation, Principal Component Analysis (PCA) for dimensionality reduction, and K-Means Clustering.
* **Highlights:** Elbow Method and Silhouette Scores for optimal `k` selection, coupled with rich 2D/3D visualizations and Radar charts to define actionable Business Personas.
